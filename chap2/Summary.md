### 1. main() 메소드의 메소드 이름 앞에는 어떤 예약어들이 들어 가나요?
public static void main(String[] args) {}
public, static, void 예약어들이 들어 간다.
* public: 접근 제어자
* static: 객체를 생성하지 않아도 호출할 수 있음
* void: return값이 없을 때 사용

### 2. main() 메소드의 매개 변수에는 어떤 값들이 들어가나요?
String[] args 가 들어간다. 
[]의 의미는 args 변수가 배열이라는 것을 말한다. 

### 3. 만약 여러분들이 만든 클래스에 main() 메소드가 없다면, java 명령어로 그 클래스를 수행할 수 있나요?
수행할 수 없다. 실행을 목적으로 한다면 main() 메소드가 필수로 있어야 한다.

### 4. System.out.println() 메소드는 어떤 용도로 사용하나요?
어떤 것을 커맨드 창에 출력할 때 사용하는 메소드이다.

### 5. System.out.print() 메소드는 System.out.println() 메소드와 어떤 차이가 있나요?
줄 바꿈의 차이가 있다. print()는 줄 바꿈 없이 연속으로 출력하게 하지만, println()은 줄 바꿈을 한다.

### 6.  /* 로 시작하고 * / 로 끝나는 사이에 있는 소스들은 어떻게 되나요?
주석 처리가 된다.

### 7. 메소드를 선언할 때 반드시 꼭 있어야 하는 세 가지는 무엇인가요?
리턴 타입, 메소드 이름, 메소드 내용이다.
